# Dev Profile Generator 
​
#### Dev-profile-generator-repo, 1/21/20
​
#### By Zayd Sarraj
​
## Description
This application takes in the a user's Github Username and provides a profile page based on it.

##Features
​
* Takes in Github Username and user's favorite color
* Generates a PDF profile page styled based on the user's color choice
* Includes information on number of repos, followers, and stars
* Includes links to user's Github page, personal blog, and google maps location
